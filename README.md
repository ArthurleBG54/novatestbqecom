# BestDeals - Thème Shopify

Généré par NOVAIA
Ce thème est prêt à être installé sur Shopify.

## Installation

1. Allez sur votre admin Shopify → Boutique en ligne → Thèmes
2. Cliquez sur "Ajouter un thème" → "Télécharger le fichier ZIP"
3. Sélectionnez ce fichier ZIP
4. Attendez l'installation
5. Cliquez "Personnaliser" pour modifier les couleurs/logo

## Structure

- layout/theme.liquid : Structure principale
- templates/ : Pages (accueil, produit, collection...)
- sections/ : Sections modulaires (hero, produits...)
- assets/ : CSS et JavaScript
- config/ : Paramètres du thème
